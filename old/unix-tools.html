*nix tools you should know

NOTES!

I've said that one of the most important things you can do is to learn a core set of *nix tools. Some you need to know how to use (at least the basics). Others you only need to know that they exist, since you can read the man page.

  • man and info
  • A powerful text editor, preferably vi (also vim, the mangled version of vi) or Emacs.
  • sed, the steam editor.
  • tr.
  • od.
  • make.

You also need to know some basic scripting techniques in bash, including pipes and I/O redirection.
